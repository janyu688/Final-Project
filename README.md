# Heading

We propose to create Mindfulness Reminder, a command-line Python program that helps users practice mindfulness throughout their day.
It will help track the health needs of the user. For ecample, weather they need to stretch or go for a run at a certain time (based on the settings that they create in the program.) It helps maintian the user's physical well-being.

## Sub-Heading

There will be different functions to regulate the different parts of the program.
Whether it to remind someone of the wellness check they should be doing at that moment (like push-ups for example) or time someone for how long they would like to do said exercise, the different functions will regulate these elements.
